breakaleg by ronaldoxgh

Fast JavaScript compiler and evaluator written in C# .NET

Breakaleg is a fast JavaScript parser and evaluator for general purpose written in C# .NET. It does not implement the DHTML model by itself. This can be done by a customized solver. Breakaleg supports classes, first class functions, prototyping, and a bunch of other cool stuff. P.S. Break-a-leg stands for "Good Luck" in theatre jargon. javascript, script, theatre, good luck, good things... you know :)

Dependencies

.NET4.0, Dynamics, MS TestFmk

Install

It's a simple VS2010 solution. You can extract the core and use it as an external module, or compile it directly into your project. As you wish.
